  WHY PROGRAMMATIC SEO FITS THIS BUSINESS EXACTLY

  The founder's journey (from Marketing Channels doc) has 4 stages:

    Stage 1  →  Feels operational pain. On LinkedIn, Twitter, founder communities.
    Stage 2  →  Tries DIY: Vapi, n8n, Zapier, ChatGPT. Breaks in 30 days.
    Stage 3  →  DIY fails in production. Googles for real help.
    Stage 4  →  Evaluates agencies. Checks website. Wants proof.

  Programmatic SEO captures Stage 3 and pre-empts Stage 2.

  The founder Googles:
    - "vapi alternative for production"           ← they just hit the wall
    - "AI receptionist for dental practice"       ← they know what they need
    - "AI agent for gohighlevel"                  ← they're already in the tool
    - "what is AI memory for voice agents"        ← they're learning
    - "AI automation for recruiting agencies"     ← they've identified the pain

  Every one of these is a page we can rank for. The content already exists in
  your business brain. We're just putting it on URLs Google can index.

  TECHNICAL IMPLEMENTATION  (Next.js — pages router)

    URL structure    Subfolders only. Never subdomains.
                     /[role]-for-[industry]        → persona pages
                     /[tool]-alternative           → comparison pages
                     /glossary/[term]              → glossary pages
                     /en-[country]/[role]-[city]   → location pages

    Data layer       All pages generated from a single TS data file.
                     getStaticPaths + getStaticProps — pre-rendered at build time.
                     No runtime API calls on page load.

    Sitemap          sitemap.xml.tsx must be extended per page type.
                     Separate sitemap: /sitemap-personas.xml, /sitemap-glossary.xml
                     Crawl budget: exclude very thin pages with noindex.

    Internal links   Hub-and-spoke model.
                     /ai-receptionist  →  links to all dental/clinic/med-spa sub-pages
                     Each spoke links back to hub + 2–3 related spokes

    No orphan pages  Every pSEO page reachable from a hub. Every hub in the sitemap.

    Schema           Service schema + FAQPage schema on every pSEO page.
                     LocalBusiness schema on location pages.
