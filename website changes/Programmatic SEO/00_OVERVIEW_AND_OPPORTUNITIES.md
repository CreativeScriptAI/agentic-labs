## Technical Notes (Next.js Specific)

- **URL structure:** Use subfolders. Pattern: `/[role]-for-[industry]` for personas, `/glossary/[term]` for glossary
- **Rendering:** Use `getStaticProps` + `getStaticPaths` to pre-generate all pages from a JSON/TS data file
- **Sitemap:** The existing `sitemap.xml.tsx` needs to be extended to include all pSEO pages
- **Internal links:** Every pSEO page links back to the main homepage CTA and to 2â€“3 related pSEO pages
- **No orphan pages:** Every page must also be linked from a hub page (e.g. `/ai-receptionist` links to all sub-pages)
- **Schema:** Each page gets `Service` schema and `FAQPage` schema
